1. **发射器上的关键信号行为**：
   - 3阶前馈均衡 (FFE)

   2. **接收器上的关键规范行为**：
      - 连续时间线性均衡器 (CTLE)
         - 决策反馈均衡器 (DFE)
            - 时钟数据恢复 (CDR)
